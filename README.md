# D3.js school

For learning D3.js with Piet's guidance.

## Setup

Do this:

```bash
npm install
```

## Run

Do this:

```bash
npm start
```

Your page will be reachable at the stated address, probably:

http://127.0.0.1:8080/

Then edit the files under [`public`](./public) and see the page change!
